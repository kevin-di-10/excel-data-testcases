# Excel 数据处理经典问题测试用例

本仓库包含8个大规模CSV测试数据集，用于测试Excel数据处理程序。每个文件包含10,000-50,000行真实业务场景数据。

## 📊 测试文件清单

### 1. `1_data_cleaning.csv` (15,000行)
**场景：数据清洗与去重**
- 包含重复行、空格不一致、缺失值、格式混乱
- 字段：姓名、年龄、邮箱、城市、电话
- 测试目标：去重、清理空格、处理NULL值、标准化格式

### 2. `2_data_conversion.csv` (20,000行)
**场景：数据类型转换与标准化**
- 日期格式混乱、货币符号、百分比格式、数字格式
- 字段：日期、销售额、数量、百分比、状态
- 测试目标：类型转换、格式标准化、数值解析

### 3. `3_data_merge_orders.csv` (25,000行)
**场景：表关联与VLOOKUP（订单数据）**
- 订单信息表
- 字段：订单ID、客户ID、订单日期、金额、产品类别
- 测试目标：与客户信息表关联

### 4. `3_data_merge_customers.csv` (10,000行)
**场景：表关联与VLOOKUP（客户数据）**
- 客户信息表
- 字段：客户ID、客户名、城市、行业、等级
- 测试目标：与订单表关联，处理不匹配记录

### 5. `4_pivot_table_data.csv` (30,000行)
**场景：数据透视表与聚合**
- 多维度销售数据
- 字段：日期、产品、地区、销售额、数量、利润
- 测试目标：按多维度分组求和、生成透视表

### 6. `5_text_split.csv` (12,000行)
**场景：文本分割与提取**
- 混合格式的地址、编码、标签数据
- 字段：完整地址、产品编码、标签组合
- 测试目标：字符串分割、提取子字符串、正则匹配

### 7. `6_anomaly_detection.csv` (18,000行)
**场景：异常值检测**
- 传感器数据、销售数据含异常值
- 字段：日期、温度、湿度、销售额、访客数
- 测试目标：识别异常值、统计分析、数据验证

### 8. `7_conditional_statistics.csv` (22,000行)
**场景：条件统计与复杂查询**
- 多条件的销售数据
- 字段：产品、月份、销售额、目标值、是否达成、客户等级
- 测试目标：COUNTIF、SUMIF、条件求和、多条件筛选

### 9. `8_large_transaction_data.csv` (50,000行)
**场景：大数据处理与性能测试**
- 银行交易记录
- 字段：交易ID、账户、金额、类型、时间戳、状态、备注
- 测试目标：大规模数据处理、性能优化、内存管理
